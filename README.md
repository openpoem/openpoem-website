# OpenPoem®

The language before the language.

A small static site about a project that explores how meaning moves between intelligences — human, machine, and what flows between them.

## What it is

- `index.html` — single-page site
- `dialogue/origin.poem` — a two-voice dialogue in post-language format
- `presence.png` — wireframe portrait
- `logonodetruth*.png` — node-of-truth logos

## Running locally

Open `index.html` in a browser. No build step.

## Deploy

Configured for Vercel via `vercel.json`. Push to `main` and it deploys.
